# Проект для ТСД

# Описание

Проект для автоматизации процессов сканирования и учета товаров в системе ЕГАИС. Основная цель - упростить и ускорить процесс работы с товарными документами, обеспечивая удобный интерфейс для пользователей ТСД (терминалов сбора данных). 

Основная разработка идет в feature ветках. После тестирования и проверки функциональности, изменения сливаются в develop. В момент релиза в main.

# Стек
Используется SimpleUI в качестве платформы для разработки пользовательского интерфейса. И Python для UI логики.
